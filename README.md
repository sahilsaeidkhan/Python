# Python-Basics

This repository contains beginner-friendly Python scripts and exercises to help you learn and practice fundamental programming concepts. It is organized into folders covering different topics, with clear examples and exercises.

## Structure

### basics/
Core Python scripts covering fundamental concepts:
- `first.py` - Basic Python syntax
- `Variables-Exercise.py` - Variables and data types
- `numbers.py` - Working with numbers
- `String.py` - String operations
- `Input.py` - Input/output operations
- `if_conditinal.py` - Conditional statements
- `forloop.py` - Loops
- `function.py` - Functions
- `Lists.py`, `Lists2.py` - Lists
- `dictionary.py` - Dictionaries
- `oops.py`, `inheritance.py` - Object-Oriented Programming
- `exceptionhandling.py` - Exception handling
- `recursion.py` - Recursion
- `Modules.py` - Modules
- `concept.py`, `practice.py` - Practice exercises

### basics/DSA/
Data Structures and Algorithms:
- `arrays.py` - Arrays
- `linkedlist.py`, `linkedlistcodebasics.py`, `doublelinkedlist.py` - Linked Lists
- `stack.py`, `stack_braces.py`, `reverse_stack.py` - Stacks
- `queue.py`, `queue_question.py` - Queues
- `tree.py` - Trees
- `simple_hash.py`, `hash_chaining.py` - Hash Tables
- `binarysearch.py` - Binary Search
- `sorting.py` - Sorting Algorithms
- `iterators.py`, `2nd_iterators.py` - Iterators
- `generator.py` - Generators
- `list_comprehension.py` - List Comprehensions
- `multithreading.py` - Multithreading

### Numpy/
NumPy library fundamentals:
- `phase-1.ipynb` - Jupyter notebook covering:
  - Array creation (`np.array`, `np.zeros`, `np.ones`, `np.full`, `np.random.random`, `np.arange`)
  - Array properties (shape, ndim, size, dtype)
  - Array reshaping and flattening

## How to Use

1. Clone or download the repository.
2. Open any script in your preferred editor (e.g., VS Code).
3. Run scripts using Python 3.x:
   ```
   python basics/<script_name.py>
   ```
4. For Numpy notebooks, set up a virtual environment:
   ```
   cd Numpy
   python -m venv venv
   source venv/bin/activate  # or venv\Scripts\activate on Windows
   pip install numpy jupyter
   jupyter notebook
   ```
